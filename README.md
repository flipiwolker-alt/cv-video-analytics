# CV Video Analytics

Учебный проект по компьютерному зрению и анализу видео. Состоит из 8 практических заданий и итоговой курсовой работы. Все задания выполняются в Google Colab — установка ничего лишнего не требуется.

---

## Что делает каждое ПЗ

### ПЗ 1 — Обработка изображений (OpenCV)
Знакомство с базовыми операциями над изображениями: изменение размера, перевод в оттенки серого, изменение яркости и контраста, размытие, поворот. Используется библиотека OpenCV.

📓 [PZ_1_OpenCV_Processing.ipynb](notebooks/PZ_1_OpenCV_Processing.ipynb)

---

### ПЗ 2 — Нарезка видео на кадры
Загружаем видеофайл на Google Drive, нарезаем его на отдельные кадры (каждый N-й кадр) и сохраняем как JPG. Это основа для всех последующих ПЗ — все они работают с этими кадрами.

📓 [PZ_2_Video_Download.ipynb](notebooks/PZ_2_Video_Download.ipynb)

---

### ПЗ 3 — Распознавание текста на кадрах (OCR)
Прогоняем кадры через EasyOCR — библиотеку для распознавания текста на изображениях. Находит и извлекает текст с субтитров, вывесок, надписей на экране. Результат сохраняется в CSV.

📓 [PZ_3_OCR_Recognition.ipynb](notebooks/PZ_3_OCR_Recognition.ipynb)

---

### ПЗ 4 — Транскрипция речи (Whisper)
Извлекаем аудиодорожку из видео и прогоняем через Whisper от OpenAI — модель для распознавания речи. Получаем текстовую расшифровку того, что говорится в видео.

📓 [PZ_4_Whisper_Audio.ipynb](notebooks/PZ_4_Whisper_Audio.ipynb)

---

### ПЗ 5 — Детекция объектов (YOLOv8)
Запускаем YOLOv8 на каждом кадре — модель обнаруживает и классифицирует объекты (люди, машины, предметы), рисует bounding boxes. Результат: CSV с координатами и классами всех найденных объектов.

📓 [PZ_5_YOLO_Detection.ipynb](notebooks/PZ_5_YOLO_Detection.ipynb)

---

### ПЗ 6 — Классификация кадров (ResNet50)
Каждый кадр прогоняем через предобученный ResNet50 (обучен на ImageNet). Модель выдаёт топ-5 вероятных классов для каждого кадра. Результат визуализируется — кадры с подписями классов.

📓 [PZ_6_ResNet_Classification.ipynb](notebooks/PZ_6_ResNet_Classification.ipynb)

---

### ПЗ 7 — Описание кадров через LLM (OpenRouter)
Отправляем кадры на FastAPI-сервер, развёрнутый на облачной ВМ (Timeweb, европейский IP). Сервер через OpenRouter вызывает мультимодальную модель `nvidia/nemotron-nano-12b-v2-vl:free`, которая описывает что изображено на каждом кадре. Результат — JSON с текстовыми описаниями + визуализация кадров с подписями.

📓 [PZ_7_LLM_API.ipynb](notebooks/PZ_7_LLM_API.ipynb)

---

### ПЗ 8 — Постобработка результатов
Финальный этап обработки данных из предыдущих ПЗ:
- **OCR**: дедупликация текста через fuzzy-matching (rapidfuzz) — убирает почти одинаковые строки
- **YOLO**: склейка детекций по временным окнам — несколько соседних кадров с одним объектом превращаются в одно "событие"
- **LLM**: анализ описаний, подсчёт самых часто встречающихся объектов
- Итоговый сводный отчёт в JSON + графики

📓 [PZ_8_PostProcessing.ipynb](notebooks/PZ_8_PostProcessing.ipynb)

---

### Курсовая — Полный пайплайн
Объединяет все ПЗ в единый сценарий: от загрузки видео до итогового отчёта. Запускается последовательно — каждый этап передаёт результаты следующему.

📓 [Coursework_Full_Pipeline.ipynb](notebooks/Coursework_Full_Pipeline.ipynb)

## Как запустить

1. Откройте [Google Colab](https://colab.research.google.com)
2. **File → Open notebook → GitHub**
3. Вставьте: `https://github.com/flipiwolker-alt/cv-video-analytics`
4. Выберите нужный ноутбук из папки `notebooks/`
5. Выполняйте ячейки по порядку сверху вниз

> ⚠️ Перед ПЗ 3–8 убедитесь что выполнено ПЗ 2 — кадры должны быть в Google Drive по пути `/MyDrive/cv-frames/кадры/`

## Технологии

| Задача | Инструмент |
|--------|-----------|
| Обработка изображений | OpenCV |
| Распознавание текста | EasyOCR |
| Транскрипция речи | OpenAI Whisper |
| Детекция объектов | YOLOv8 (Ultralytics) |
| Классификация | ResNet50 (PyTorch) |
| LLM описания | nvidia/nemotron-nano-12b-v2-vl via OpenRouter |
| API сервер | FastAPI + Docker (Timeweb Cloud) |
| Постобработка | pandas, rapidfuzz |
